# Git tips

> Clone source code từ github/gitlab về máy

```bash
git clone <source>
```

> Kiểm tra trạng thái các file

```bash
git status
```

> Kiểm tra thay đổi của file

```bash
git diff <tên file>
```

> Thêm file vào commit

```bash
git add <tên file>
```

> Commit

```bash
git commit -m <message>
```

> Push code lên remote branch

```
git push origin <tên branch>
```
